# Code of Conduct

We follow the [CNCF Code of Conduct](https://github.com/cncf/foundation/blob/main/code-of-conduct.md).
It applies to all project spaces and activities, including issues, pull requests, reviews, and other
project-related communication.

## Reporting

Report conduct concerns privately to [oss@ollygarden.com](mailto:oss@ollygarden.com). Do not open a
public issue or pull request to report an incident.

Reports are handled by the OllyGarden maintainers, not by the CNCF Code of Conduct Committee. We
will protect the privacy and safety of reporters and affected community members to the extent
possible while reviewing and resolving a report.
